## Monthly Budget
> This project requires you to download mariadb, python, node.js, on local
1. Change directory into the project
   ```
   cd monthly_budget
   ```
2. Run shell commands to set up mariadb database
   ```
   bash command.sh
   ```
3. Run back-end
   ```
   cd server
   flask run
   ```
4. Run front-end
   ```
   npm init
   npm start
   ```
